# Windows Server 2022 Home Lab

## Overview

This project documents the creation of a virtualized Windows Server 2022 environment using Oracle VirtualBox.

The goal of this lab is to practice enterprise system administration and cybersecurity concepts.

## Infrastructure

Domain Name:
- corp.local

Virtual Machines

- DC01
  - Windows Server 2022
  - Active Directory Domain Services
  - DNS

- CLIENT01
  - Windows 10 Enterprise Evaluation (archived)
  - Domain Joined

- CLIENT02
  - Windows 11 Enterprise Evaluation
  - Domain Joined
